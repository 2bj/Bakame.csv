Installation

System Requirements

You need PHP >= 5.4.0 and the mbstring extension to use Bakame\Csv but the latest stable version of PHP is recommended.

Installation

Bakame.csv is available on Packagist and can be installed using Composer:


{
    "require": {
        "bakame/csv": "4.*"
    }
}


<php

//include the Composer autoloader
require 'vendor/autoload.php';

Downloading the library

This project is also available for download on GitHub. Visit the releases page, select the version you want, and click the preferred archive type download button.
